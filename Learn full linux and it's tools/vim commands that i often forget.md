(normal mode)
x -> Delete character at cursor

I (capital 'i' not 'L') -> Insert at beginning of line

a -> Append at cursor

A -> Append at end of line

w -> next word

2w -> go two words forward (you can use any number instead of 2)

d$ or D -> delete to the end of line (from where cursor is. not like 'dd' which deletes the whole line regardless of the cursor postion)

d2w -> delete two words (you can use any number instead of 2)

3e -> go to end of 3rd word ahead (ahead means it applies from where the cursor is)

$ -> go to end of the current line

2dd -> delete 2 lines (you can use any number instead of 2)

//the numbering logic is similar for all commands. you'll notice the pattern soon

`^` -> go to beginning of text on current line (it's used because there could have space or tab used on that line and the first few character might be blank space)

u -> undo last changes

U -> undo changes on the entire line

0 (zero) -> go to beginning of current line (even if it has blank spaces at the beginning)

ctrl + r -> redo changes

(visual mode) # can be entered with 'v' key from normal mode
visual mode is used for selecting / highlighting text
